Morpion - Projet Développement Web
Ce projet est ma version du classique jeu de Morpion. Je l'ai codé pour mettre en pratique la manipulation dynamique du DOM en JavaScript et renforcer ma logique algorithmique de première année.

💡 Ce que j'ai appris et mis en œuvre
Logique de jeu : J'ai dû implémenter les conditions de victoire et la gestion des tours de jeu, ce qui m'a obligé à bien structurer mes fonctions pour éviter les répétitions.

Interaction utilisateur : J'ai créé une gestion d'événements fluide pour passer du menu de sélection au plateau de jeu, incluant les écrans de fin de partie.

Mode "vs Robot" : C'était le défi le plus intéressant. J'ai dû apprendre à concevoir une petite routine pour que le robot joue ses coups de manière cohérente.

Vanilla JS : J'ai volontairement choisi de ne pas utiliser de bibliothèque externe. Cela m'a permis de mieux comprendre comment JavaScript communique avec le HTML et le CSS.

🛠 Techno
HTML/CSS : Mise en page des interfaces.

JavaScript : Logique complète, gestion des scores et des états du jeu.

🚀 Pourquoi ce projet ?
Ce Morpion est une étape importante dans mon apprentissage. Il illustre ma capacité à partir d'une idée simple et à la structurer en un programme fonctionnel, propre et réactif. C'est une base que je compte faire évoluer avec de nouvelles fonctionnalités prochainement.
